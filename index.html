Lintcode学习笔记之二叉树先序遍历

发表于 2017-08-03 | 分类于 算法学习

题目描述

Given a binary tree, return the preorder traversal of its nodes’ values.

例子

return [1,2,4,5,3].

思考

1. 二叉树的遍历是可以用递归和非递归的方法进行解答的。
2. 先序遍历的遍历顺序是根节点，左孩子节点，右孩子结点。

代码实现

/*
 * Definition of TreeNode:
 * public class TreeNode {
 * public int val;
 * public TreeNode left, right;
 * public TreeNode(int val) {
 * this.val = val;
 * this.left = this.right = null;
 * }
 * }
 */

//采用递归的方法，比较简单
public class Solution{
   //result的定义一定要在方法体之外，
   ArayList<Integer> result=new ArrayList<Integer>();
   public ArrayList<Integer> preorderTraversal(TreeNode root){
 if(root==null){return result;} //常规套路验证是否根节点为空
 result.add(root.val); //集合保存根节点的值
 preorderTraversal(root.left); //遍历左子树
 preorderTraversal(root.right);//遍历右子树
 return result;
  }
}

   //采用非递归的方法： 不停地找左子树的左子树，直到找到最左端的叶子结点，
   采用stack的存储根结点(每棵小树里面的)，再来找右子树，而遍历右子树用的还是左子树的套路。 
   public class Solution{
   public ArrayList<Integer> preorderTraversal(TreeNode root){
   ArayList<Integer> result=new ArrayList<Integer>();
   Stack<TreeNode> stack=new Stack<TreeNode>();
   TreeNode curt=root;
   if(root==null){return result;} 
   while(curt!=null || !stack.empty()) {
   while(curt!=null){ 
   result.add(curt.val); //保存根结点
   stack.push(curt); //将根结点放在栈当中
   curt=curt.left;   //继续找根节点的左孩子结点
}

   //这里出现有根结点但是没有左孩子的情况 。所以curt是空的。
   // 有两种情况：1.有右孩子 2.无右孩子
   curt=stack.pop();  //弹出该无左孩子的根结点
   curt=curt.right;   //找到右孩子继续遍历 
 }
      return result;
   }
}  

// 非递归方法(version 2) 该方法着眼于每颗由根节点、左孩子、右孩子构成的小树。

public ArrayList<Integer> preorderTraversal(TreeNode root){
 if(root==null){return result;} //常规套路验证是否根节点为空
 ArayList<Integer> result=new ArrayList<Integer>();

//分治法(Divide & Conquer) 将整棵树看作一个整体，分别是根节点，左子树和右子树
public class Solution{
  public ArrayList<Integer> preorderTraversal(TreeNode root){
  ArrayList<Integer> result=new ArrayList<Integer>();  
  if(root==null){return result;}

  result.add(root.val);
  ArrayList<Integer> left=preorderTraversal(root.left); //左子树
  ArrayList<Integer> right=preorderTraversal(root.right);//右子树

  result.addAll(left);  //合并左子树结果
  result.addAll(right); //合并右子树结果

  return result;
 }

Lintcode学习笔记之Balanced Binary Tree

发表于 2017-07-31 | 分类于 算法学习

问题描述

Given a binary tree, determine if it is height-balanced.
For this problem, a height-balanced binary tree is defined as a binary tree in which the depth of the two subtrees of every node never differ by more than 1.

例子

Given binary tree A = {3,9,20,#,#,15,7}, B = {3,#,20,15,7}

思考

分支条件
1.空节点的二叉树
2.在根节点的下的左右子树的深度相差已经>1
3.在2条件不成立，继续遍历分别以左右子树作为根节点的情况

代码展示

/**
 * //定义一棵树 
 * Definition of TreeNode:
 * public class TreeNode {
 * public int val;
 * public TreeNode left, right;
 * public TreeNode(int val) {
 * this.val = val;
 * this.left = this.right = null;
 * }
 * }
 */

public class Solution {
/**
 * @param root: The root of binary tree.
 * @return: True if this Binary tree is Balanced, or false.
 */
 // TotalTime :1947ms
public boolean isBalanced(TreeNode root) {
// write your code here
if(root==null) {return true;} // 根节点为空的情况返回 true
int left=depth(root.left);    // 通过depth(TreeNode)方法得到左子树的深度
int right=depth(root.right);  // 同理
if(left-right>1||left-right<-1){ //两颗子树的深度相差超过1
return false;
}else{
return isBalanced(root.left)&&isBalanced(root.right); // 继续与左右子树作为根节点进行遍历
}

}
private int depth(TreeNode node){
if(node==null){
return 0;
}else if(node.left==null&&node.right==null){
return 1;
}else{
int left=depth(node.left);
int right=depth(node.right);
return 1+(left>right?left:right);
}
}
}

Lintcode学习笔记之Add Two Numbers

发表于 2017-07-29 | 分类于 算法学习

问题描述

You have two numbers represented by a linked list, where each node contains a single digit. The digits are stored in reverse order, such that the 1’s digit is at the head of the list. Write a function that adds the two numbers and returns the sum as a linked list.

例子

Given 7->1->6 + 5->9->2. That is, 617 + 295.

Return 2->1->9. That is 912.

Given 3->1->5 and 5->9->2, return 8->0->8.

思考

1.如果两个数的位数不同怎么办？
答：设置两个分支，分别是两个数在特定的位上都有数和特定的位置只有其中一个数（那个更长的）的情况。
2.怎么处理位数溢出的要求？
答：给定一个变量carry记录变化，把它放在链表的最后一位。
3.怎样设置链表能显示计算结果是倒置的效果 ？
答：按照链表的顺序相加即可。

代码实现

/**
 * Definition for singly-linked list.
 * public class ListNode {
 * int val;
 * ListNode next;
 * ListNode(int x) {
 * val = x;
 * next = null;  
 * }
 * }
 */

public class Solution {

/**
 * @param l1: the first list
 * @param l2: the second list
 * @return: the sum list of l1 and l2 
 */
public ListNode addLists(ListNode l1, ListNode l2) {
// write your code here
if(l1==null&&l2==null){ //判断链表是否为空
return null;
}

ListNode head=new ListNode(0);//设置一个空的链表头，这是一个套路，删除链表的时候也会用到，用以避免处理表头是否需要删除的问题。
ListNode point=head;  //用point作为具体操作的结点
int carry=0;          //定义进位数字标志位
while(l1!=null&&l2!=null){
int sum=l1.val+l2.val+carry;
point.next=new ListNode(sum%10);  //保留个位
carry=sum/10;                     //保留个位之外的数字
l1=l1.next;
l2=l2.next;
point=point.next;
}

while(l1!=null){                 //如果l1是那个更长的链表则用l1进行相加
int sum=l1.val+carry;
point.next=new ListNode(sum%10);
carry=sum/10;
l1=l1.next;
point=point.next; 
}
while(l2!=null){                //如果l2是那个更长的链表则用l1进行相加
int sum=l2.val+carry;
point.next=new ListNode(sum%10);
carry=sum/10;
l2=l2.next;
point=point.next; 
}

if(carry!=0){                  //进位数标志位不为0时说明要再设置一位，数值就为carry的值
point.next=new ListNode(carry);
}
return head.next;
}
}
