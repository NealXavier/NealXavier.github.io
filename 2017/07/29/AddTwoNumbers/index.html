Lintcode学习笔记之Add Two Numbers

发表于 2017-07-29 | 分类于 算法学习

问题描述

You have two numbers represented by a linked list, where each node contains a single digit. The digits are stored in reverse order, such that the 1’s digit is at the head of the list. Write a function that adds the two numbers and returns the sum as a linked list.

例子

Given 7->1->6 + 5->9->2. That is, 617 + 295.

Return 2->1->9. That is 912.

Given 3->1->5 and 5->9->2, return 8->0->8.

思考

1.如果两个数的位数不同怎么办？
答：设置两个分支，分别是两个数在特定的位上都有数和特定的位置只有其中一个数（那个更长的）的情况。
2.怎么处理位数溢出的要求？
答：给定一个变量carry记录变化，把它放在链表的最后一位。
3.怎样设置链表能显示计算结果是倒置的效果 ？
答：按照链表的顺序相加即可。

代码实现

/**
 * Definition for singly-linked list.
 * public class ListNode {
 * int val;
 * ListNode next;
 * ListNode(int x) {
 * val = x;
 * next = null;  
 * }
 * }
 */

public class Solution {

/**
 * @param l1: the first list
 * @param l2: the second list
 * @return: the sum list of l1 and l2 
 */
public ListNode addLists(ListNode l1, ListNode l2) {
// write your code here
if(l1==null&&l2==null){ //判断链表是否为空
return null;
}

ListNode head=new ListNode(0);//设置一个空的链表头，这是一个套路，删除链表的时候也会用到，用以避免处理表头是否需要删除的问题。
ListNode point=head;  //用point作为具体操作的结点
int carry=0;          //定义进位数字标志位
while(l1!=null&&l2!=null){
int sum=l1.val+l2.val+carry;
point.next=new ListNode(sum%10);  //保留个位
carry=sum/10;                     //保留个位之外的数字
l1=l1.next;
l2=l2.next;
point=point.next;
}

while(l1!=null){                 //如果l1是那个更长的链表则用l1进行相加
int sum=l1.val+carry;
point.next=new ListNode(sum%10);
carry=sum/10;
l1=l1.next;
point=point.next; 
}
while(l2!=null){                //如果l2是那个更长的链表则用l1进行相加
int sum=l2.val+carry;
point.next=new ListNode(sum%10);
carry=sum/10;
l2=l2.next;
point=point.next; 
}

if(carry!=0){                  //进位数标志位不为0时说明要再设置一位，数值就为carry的值
point.next=new ListNode(carry);
}
return head.next;
}
}
