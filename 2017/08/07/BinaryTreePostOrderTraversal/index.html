Lintcode学习笔记之二叉树后序遍历

发表于 2017-08-07 | 分类于 算法学习

题目描述

Given a binary tree, return the postorder traversal of its nodes’ values.

例子

return [3,2,1].

思考

  1. 本质上和前后序遍历没有什么区别。
  2. 增加一个pre结点作为标志，标记之前的结点的位置，为了表示右孩子到底有没有遍历完从而决定要不要从栈当中弹出根节点。此外其实都是之前的老套路，还是先找到叶子结点的最左侧。

代码实现(不展示递归方法)

 // Iteration :
下面在弹栈的时候需要分情况一下：
1）如果当前栈顶元素的右结点存在并且还没访问过（也就是右结点不等于上一个访问结点），
那么就把当前结点移到右结点继续循环；
2）如果栈顶元素右结点是空或者已经访问过，那么说明栈顶元素的左右子树都访问完毕，
应该访问自己继续回溯了。
下面列举一下代码：
     public class Solution{
 public ArrayList<Integer> postorderTraversal(TreeNode root) {  
 ArrayList<Integer> postOrder=new ArrayList<Integer>(); 
 Stack<TreeNode> stack=new Stack<TreeNode>(); 
 TreeNode pre=null,curr=root; //初始化当前的结点为root而pre(之前结点)为null

while(curr!=null||!stack.empty()){  //遍历结束的条件是栈的值为空或者当前结点为空
 if(curr!=null){            // 为了找到叶子结点的最左侧
   stack.push(curr);
   curr=curr.left;
 }else{                     // 已经找到最左端的叶子结点
   TreeNode peekNode=stack.peek();  // 不确定其还有没有右孩子所有不用pop方法
   if(peekNode.right!=null&&pre!=peekNode.right){ // 如果该结点有右孩子并且它还没有被遍历过
   curr=peekNode.right;  //将其右孩子替换成当前结点
   }else{   //有两种情况 1. 其没有右孩子 2.其有右孩子 但是已经被遍历过了
   stack.pop();     // 弹出该结点 
   postOrder.add(peekNode.val);    
   pre=peekNode;   // 修改pre(之前结点) ，将该结点作为当前结点 
 }
    }
  }
return postOrder;
    }
 }


//version2 :Iteration
 思路：
（1）如果pre的左孩子或者右孩子是cur，那么说明遍历在往下走，按访问顺序继续，
 即如果有左孩子，则是左孩子进栈，否则如果有右孩子，则是右孩子进栈，
 如果左右孩子都没有，则说明该结点是叶子，可以直接访问并把结点出栈了。
 (2）如果反过来，cur的左孩子是pre，则说明已经在回溯往上走了，
 但是我们知道后序遍历要左右孩子走完才可以访问自己，
 所以这里如果有右孩子还需要把右孩子进栈，否则说明已经到自己了，可以访问并且出栈了。
（3）如果cur的右孩子是pre，那么说明左右孩子都访问结束了，可以轮到自己了，
 访问并且出栈即可。 
 public class Solution{
  public ArrayList<Integer> postorderTraversal(TreeNode root) {  
  ArrayList<Integer> res = new ArrayList<Integer>();  
  if(root == null){ return res;}  
  Stack<TreeNode> stack = new Stack<TreeNode>();  
  stack.push(root);  
  TreeNode pre = null;  
  while(!stack.Empty()) {  
  TreeNode cur = stack.peek(); //找到栈顶的值  
  if(pre==null || pre.left==cur || pre.right==cur){ 
  //1.为null还没开始开始遍历  2，3 表示 二叉树在往下遍历的过程中   
        if(cur.left!=null) {    
            stack.push(cur.left);  
        } else if(cur.right!=null){  
            stack.push(cur.right);  
        }else{               // 没有左右孩子说明已经是在叶子结点，可以弹出栈
            cur=stack.pop();
            res.add(cur.val);  
        }  
    }else if(cur.left==pre && cur.right!=null){  
   // 说明二叉树在回溯的过程中并且当前右孩子结点还没有遍历过
        stack.push(cur.right);  
    }else{ // 1.cur没有右孩子  2.cur有右孩子结点但是已经遍历完毕  
        res.add(cur.val);  
        stack.pop();  
    }  
    pre = cur;    //修改当前pre(之前结点)的值 
}  
return res;  
 } 
}
