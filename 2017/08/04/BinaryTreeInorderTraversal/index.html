Lintcode学习笔记之二叉树中序遍历

发表于 2017-08-04 | 分类于 算法学习

问题描述

Given a binary tree, return the inorder traversal of its nodes’ values.

例子

Given binary tree {1,#,2,3},

return [1,3,2].

思考

  1. 非递归方法和先序遍历相差无几，还是需要找到叶子结点的最左端。
  2. 还是可以用分治法来解决

代码实现

/**
 * Definition of TreeNode:
 * public class TreeNode {
 * public int val;
 * public TreeNode left, right;
 * public TreeNode(int val) {
 * this.val = val;
 * this.left = this.right = null;
 * }
 * }
 */

// version1 : 递归(recrusion) 
public class Solution{
    ArrayList<Integer> inorder=new ArrayList<Integer>();
    public ArrayList<Integer> inorderTraversal(TreeNode root) {
    // write your code here
    if(root==null){ return inorder;}

    inorderTraversal(root.left);
    inorder.add(root.val);
    inorderTraversal(root.right);
    return inorder;
   }
}

左孩子结点为空时的情况如下：

//version2 :  非递归（迭代）
 public class Solution{
 public ArrayList<Integer> inorderTraversal(TreeNode root) {
 ArrayList<Integer> res=new ArrayList<Integer>();
 Stack<TreeNode> stack=new Stack<TreeNode>();
 while(root!=null||!stack.empty()){
    if(root!=null){
        stack.push(root);
        root=root.left;
    }else{
        root=stack.pop();
        res.add(root.val);
        root=root.right;
    }
}
    return res;
  }
}

//version3 :  分治法(Divide & Conquer)
public class Solution{

 public ArrayList<Integer> inorderTraversal(TreeNode root){
     ArrayList<Integer> inorder=new ArrayList<Integer>(); //inorder的值要放在方法体里面
     if(root==null){ return inorder;}
     //Divide 
     ArrayList<Integer> left=inorderTraversal(root.left);
     ArrayList<Integer> right=inorderTraversal(root.right);

     //Conquer
     inorder.addAll(left);
     inorder.add(root.val);
     inorder.addAll(right);
     return inorder; 
 }
}
